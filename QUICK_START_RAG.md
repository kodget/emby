# 🚀 Quick Start - RAG System

## 3 Commands to Get Started

### 1. Install Dependencies (2 minutes)

```bash
cd backend
.\install_rag.bat
```

### 2. Process Your Slides (5-10 minutes)

```bash
python manage.py process_slides_rag --all
```

### 3. Restart Server

```bash
python manage.py runserver
```

**Done!** Open your browser and test the AI panels.
